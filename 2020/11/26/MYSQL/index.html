修改数据库

1.修改数据库的输入数据类型
格式：alter table （表名） modify （要修改的内容）
2.增添数据库中的一个输入项
格式：alter table （表名）add （要增加的内容）
3.更改原本的数据库项目名字
格式： alter table （表名）change column（要修改的） （修改结果）（数据类型）
4.修改表名
格式：alter table （原本表名） rename （现在想要的表名）
这两个都是属于结构设计范畴的。
