  • asort() - 根据关联数组的值，对数组进行升序排列

    运行结果：Array ( [Joe] => 43 [Ben] => 37 [Peter] => 35 )

  • krsort() - 根据关联数组的键，对数组进行降序排列

运行结果：Array ( [Peter] => 35 [Joe] => 43 [Ben] => 37 )

四、PHP 循环 - While 循环

  • while - 只要指定的条件成立，则循环执行代码块

  • do…while - 首先执行一次代码块，然后在指定的条件成立时重复这个循环

  • for - 循环执行代码块指定的次数

    注释：上面的初始值和增量参数可为空，或者有多个表达式（用逗号分隔）

  • foreach - 根据数组中每个元素来循环代码块

    （1） 每进行一次循环，当前数组元素的值就会被赋值给 $value 变量（数组指针会逐一地移动），在进行下一次循环时，您将看到数组中的下一个值。

（2）每一次循环，当前数组元素的键与值就都会被赋值给 $key 和 $value 变量（数字指针会逐一地移动），在进行下一次循环时，你将看到数组中的下一个键与值。
