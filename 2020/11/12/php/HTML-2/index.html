HTML学习笔记(二)

一、HTML 标题

1.HTML 标题

标题（Heading）是通过 < h1> - < h6> 标签进行定义的。

< h1> 定义最大的标题。 < h6> 定义最小的标题。

2.HTML 水平线

< hr /> 标签在 HTML 页面中创建水平线,用于分割各部分内容的。

3.HTML 注释

可以将注释插入 HTML 代码中，这样可以提高其可读性，使代码更易被人理解。浏览器会忽略注释，也不会显示它们。

< !– 注释 –>

二、HTML 段落

1、HTML 段落

段落是通过 < p> 标签定义的。

< p>hello < /p>

< p>大家好< /p>

2、HTML 换行

在不产生一个新段落的情况下进行换行，使用 < br> 标签：

< p>hello< br>大家< br>好呀< /p>
